# AI 기반 여름 패션 브랜드 광고 패키지 제작 파이프라인

본 프로젝트는 생성형 AI 멀티모달 도구(Midjourney, Runway, Suno)를 활용하여 가상의 여름 테크니컬 스트릿 패션 브랜드 **'COOL SPOT'**의 9초 광고 영상을 기획하고 제작한 전체 파이프라인 프로세스를 기록한 문서입니다.

---

## 1. 브랜드 아이덴티티 및 캠페인 정의

### 브랜드 아이덴티티
- **브랜드명**: 쿨스팟 (COOL SPOT)
- **타겟**: 10~20대 (트렌디/스트릿 패션 크루 및 무더운 여름에도 힙한 스타일을 고수하는 트렌디 피플)
- **톤앤매너**: 시네마틱 아이스(Ice), 청량한 블루/실버 컬러웨이, Y2K 스포티 스트릿, 빠르고 감각적인 비트 매칭
- **USP**: 입는 순간 체감 온도를 낮춰주는 초냉감 기술 원단과 미래지향적 가상 도시(사이버펑크) 실루엣의 결합
- **캠페인 목표**: 브랜드 출시 인지 및 시각적 청량감 각인
- **핵심 메시지(한 문장)**: “이 여름, 가장 시원한 곳은 네가 서 있는 바로 그곳.”

## 4. 프롬프트 수정 전/후 기록

## 씬 2 프롬프트 개선 로그(요약)

- **수정 전 의도**: “시원한 여름 옷을 입은 20대 모델, 하이테크 패션, 도심 배경”
- **문제**: '시원한 옷'이라는 표현이 모호하여 일반적인 파란색 반팔티를 입은 평범한 학생처럼 나옴. 브랜드가 추구하는 가상 도시(사이버펑크) 거리 패션 감성과 시각적인 차가움이 전혀 표현되지 않음.
- **수정 후 변경**: “미래지향적인 은색 메탈과 얼음색 원단의 기술적인 스트릿 웨어를 입은 20대 모델의 얼굴과 상반신을 아래에서 올려다보는 구도, 가상 도시 패션 스타일, 옷 표면에 차가운 물방울이 맺혀 얼어붙는 디테일, 모델 주변으로 하얀 냉기와 연기가 은은하게 피어오르는 효과, 어두운 도심 밤거리와 푸른 네온 조명 배경, 극도로 정밀한 원단 질감 표현, 16:9 화면 비율”로 프롬프트 조정
- **결과 변화**: 모호하던 의상이 은색 메탈감의 독특한 기능성 스트릿 웨어로 정교화되었고, 냉기와 얼어붙는 물방울 표현이 추가되어 브랜드가 원하는 시각적 청량감과 톤앤매너 일관성이 대폭 개선됨.

## 6. 전체 제작 파이프라인 설계 구조

본 프로젝트는 생성형 AI 도구를 단순 나열 방식이 아닌, 단계별 의존성을 가진 파이프라인 구조로 설계하였다.

### 전체 흐름
기획 → 프롬프트 설계 → 이미지 생성(Midjourney) → 영상 변환(Runway) → 오디오 생성(Suno) → 편집 및 통합(CapCut)

### 설계 특징
- 각 단계 결과물이 다음 단계 입력으로 사용되는 구조 (Sequential Pipeline)
- 이미지 생성 결과를 기준으로 영상 모션이 결정되는 구조
- 사운드는 영상 컷 편집 속도(리듬)에 맞춰 후반에 적용되는 구조

### 설계 목적
- 멀티모달 AI 결과물 간 일관성 유지
- 제작 과정의 반복 비용 최소화
- 각 도구의 역할을 분리하여 효율 극대화

## 7. 사용 기술 원리 설명

본 프로젝트는 생성형 AI의 멀티모달 기술 구조를 기반으로 한다.

### Text-to-Image (Midjourney)
텍스트 프롬프트를 기반으로 시각적 이미지를 생성하는 기술로,
브랜드의 질감(얼음, 금속, 네온)을 시각적으로 구현하는 데 사용하였다.

### Image-to-Video (Runway Gen-3)
정지 이미지를 입력으로 받아 카메라 모션, 깊이감, 입자 효과 등을 추가하여 영상화하는 기술이다.
본 프로젝트에서는 줌인, 얼음 파편, 냉기 효과 등을 구현하는 데 사용하였다.

### Text-to-Audio (Suno AI)
텍스트 기반으로 음악 구조를 생성하는 모델로,
영상의 분위기와 동기화된 하우스 기반 사운드를 자동 생성하였다.

## 9. 톤앤매너 및 스타일 일관성 유지 전략

전체 영상은 “Ice Cyber Street” 스타일을 기준으로 통일하였다.

### 고정 키워드
- ice cold lighting
- neon blue / silver palette
- cyberpunk street environment
- cinematic motion blur

### 적용 방식
- 모든 Midjourney 프롬프트에 동일한 스타일 키워드 반복 적용
- Runway 영상에서도 동일 색감 유지
- Suno 음악은 120~130BPM 하우스 계열로 고정

이를 통해 시각/청각 요소 간 스타일 불일치를 방지하였다.

## 11. 에셋 표준화 및 생성 파라미터 설계

본 프로젝트는 시각적 일관성을 유지하기 위해 생성 단계에서 다음과 같은 표준 규격을 적용하였다.

### 해상도 및 비율
- 이미지: 1024x1792 (9:16) 또는 16:9 통일
- 영상: 1920x1080 (Full HD, 30fps 고정)

### 색감 기준
- 메인 팔레트: Ice Blue / Silver / Neon Cyan
- 보조 색상: Deep Black / Metallic Gray

### Midjourney 생성 파라미터
- --ar 16:9
- --stylize 200~400 (과도한 변형 방지)
- --seed 고정값 사용 (스타일 일관성 유지)

### 목적
- 씬 간 시각적 불일치 방지
- 브랜드 아이덴티티 유지
- 결과물 재현 가능성 확보

## 12. 도구 선택 기준 및 우선순위

본 프로젝트는 다음 기준을 우선순위로 도구를 선택하였다.

- Suno 선택 이유: 빠른 음악 생성과 톤 자동 맞춤

### 결론
비용보다 “브랜드 시각 완성도”를 최우선 기준으로 설정하였다.

## 13. 캐릭터 및 스타일 일관성 제어 방법

프로젝트에서는 다음과 같은 방식으로 스타일 일관성을 유지하였다.

### 1. 레퍼런스 이미지 고정 (--sref)
- 동일 스타일 레퍼런스를 모든 씬에 반복 적용

### 2. Seed 값 고정
- 동일한 시각 구조 유지

### 3. 프롬프트 키워드 반복
- ice / neon / cyber street / metallic texture 고정 사용

### 4. 후처리 보정 (CapCut)
- 색온도: blue tone +20%
- 대비: +10% 강화
- 노이즈: cinematic grain 추가

### 결과
도구 간 결과 차이를 최소화하고 브랜드 시각 통일성을 확보하였다.

## 14. 멀티모달 결과 통합 보정 전략

서로 다른 AI 도구 결과물은 다음 기준으로 통합하였다.

### 문제 발생 요소
- 이미지 vs 영상 색감 차이
- 프레임 속도 차이
- 밝기 및 대비 불일치

### 보정 방법
- CapCut에서 LUT 기반 색 보정 적용
- 전체 영상에 동일 필터 적용
